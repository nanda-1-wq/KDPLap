# KDP Lab — Web App UI Kit

High-fidelity recreation of the KDP Lab marketing landing page and core app screens.

## Files
- `index.html` — Interactive landing page prototype (click-through)
- `Nav.jsx` — Navbar component (scrolled + transparent states)
- `Hero.jsx` — Hero section with mock dashboard card
- `Features.jsx` — How-it-works steps + feature highlight + feature grid
- `Dashboard.jsx` — Post-login dashboard (book list, stats)
- `Generator.jsx` — Word search generator screen

## Design Width
1280px desktop. Responsive down to 375px mobile.

## Fonts
Loaded from Google Fonts:
- `JetBrains Mono` 400/500/700/800 — display/headings
- `DM Sans` 300/400/500 — body

## Colors
See `../../colors_and_type.css` for the full token set.

## Usage
Open `index.html` in a browser. Click "Get Started Free" or "Log in" to navigate between screens.
